# Contacts-App-C-Sharp-Windows-Forms-Application
A C# Windows Forms application for managing contacts using a clean 3-tier architecture (Presentation, Business Logic, Data Access). The system provides full CRUD functionality with SQL Server integration, allowing users to add, edit, delete, and view contacts efficiently.

It includes country management, image handling for contacts, and an intuitive UI built with event-driven programming principles. The project demonstrates solid OOP design, separation of concerns, and real-world database-driven application structure.
